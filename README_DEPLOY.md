# drlib GitHub Pages landing page

This folder contains a one-page project website for the existing repository:

- Repository: `https://github.com/SinaMehboodi/drlib`
- Intended Pages URL: `https://sinamehboodi.github.io/drlib/`

## Add it to the existing repository

1. Copy `index.html` to the **root** of the `drlib` repository.
2. Keep the existing `docs_assets/` directory. The page uses those figures through relative paths.
3. Commit and push the change.
4. On GitHub, open **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch **main** and folder **/(root)**, then save.

GitHub will publish the page after the Pages workflow finishes.

## Files in this preview package

- `index.html` — website file to add to the repository root
- `docs_assets/` — local copies of the figures used for offline preview
- `preview-desktop.png` and `preview-mobile.png` — rendered previews

The website is static and has no build step or external JavaScript dependency.
